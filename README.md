# NETFLIX-CLONE-MUI

Netflix Clone App built with React and MUI. 

## Built With

- React
- MUI
- Vite
- TMDb API

## User Stories
- User can search for movies and TV shows on TMDb
- User can the see upcoming and trending movies. Data updates weekly
